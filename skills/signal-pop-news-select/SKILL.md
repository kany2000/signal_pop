# Signal Pop 新闻稿选择

## 用途

从 `filtered_news_*.json` 中挑选新闻，生成结构化播报脚本。分两种稿型，按规则输出。

## 日期规则

**脚本日期 = 出稿日 + 1 天**

信号弹新闻播报和文字稿的日期统一用脚本日期。

例：出稿日 `20260718`（周日）→ 脚本日期 `20260719`（周一），播报"7月19日周一"

### 完整对应

| 出稿日 | 脚本日期（播报/文字稿日期） | 稿型 |
|--------|---------------------------|------|
| 周日 | **周一** | 平日稿 |
| 周二 | **周三** | 平日稿 |
| 周四 | **周五** | 平日稿 |
| 周五 | **周六** | 周末稿 |

## 稿型规则

### 平日稿（周日/周二/周四）

| 项目 | 值 |
|------|-----|
| 总条数 | **10 条** |
| 国内 | **6 条** |
| 国际 | **4 条** |
| 选题类别 | 经济、娱乐、时事、科技、民生 |
| 类别分布 | 按当天热点灵活分配，不强制平均 |

### 周末稿（周五）

| 项目 | 值 |
|------|-----|
| 总条数 | **15 条** |
| 国内 | **10 条** |
| 国际 | **5 条** |
| 选题类别 | 经济、娱乐、时事、科技、民生 |
| 类别分布 | 按当周热点灵活分配 |
| 内容要求 | **不能与本周平日稿重复**，选本周最受关注的热点综合 |
| 主播评论 | 从 15 条中选 **8 条** 加 AI 主播观点评论，优先选有争议性/话题性/与受众相关的新闻加评论 |

### 稿型对应表

| 出稿日 | 脚本日期（播报日期） | 稿型 | 说明 |
|--------|-------------------|------|------|
| 周日 | **周一** | 平日稿 | 10条，播报日期=周一 |
| 周二 | **周三** | 平日稿 | 10条，播报日期=周三 |
| 周四 | **周五** | 平日稿 | 10条，播报日期=周五 |
| 周五 | **周六** | 周末稿 | 15条本周热点综合，播报日期=周六 |

## 排除规则

- **政治敏感话题** — 涉及领土主权、民族宗教、敏感历史事件等
- 博彩/广告/竞猜内容
- 全英文标题（未翻译）
- 明显旧闻（日期早于 7 天前）
- 低质内容（摘要过短 <50 字、标题党、空洞无信息）
- 体育赛事、财经行情等不适合口播的类型

## 选择标准

| 维度 | 说明 |
|------|------|
| 时效性 | 优先选 48h 内的新闻 |
| 话题多样性 | 覆盖经济/娱乐/时事/科技/民生，不扎堆 |
| 来源均衡 | 分散不同 RSS 源 |
| 内容质量 | 摘要完整含数据/细节 |
| 热点程度 | 涉及大公司、政策变化、重大事件、社会广泛关注 |
| 适合播报 | 叙事型新闻，适合口播 |

## 输出格式

### 平日稿

```json
[
  {
    "title": "新闻标题",
    "link": "原文链接",
    "source": "来源名称",
    "published": "2026-07-16T10:30:00",
    "summary": "摘要正文（100字以内）",
    "category": "domestic",
    "topic": "时事",
    "selection_reason": "选择理由"
  }
]
```

### 周末稿（含主播评论）

```json
[
  {
    "title": "新闻标题",
    "link": "原文链接",
    "source": "来源名称",
    "published": "2026-07-16T10:30:00",
    "summary": "摘要正文（100字以内）",
    "category": "domestic",
    "topic": "时事",
    "selection_reason": "选择理由",
    "opinion": "主播评论观点（选中的8条有值，其余为空）"
  }
]
```

周末稿 15 条中选 **8 条** 加 `opinion` 字段，优先选有争议性、话题性、与受众相关的新闻评论。

### 主播评论要求

- **有建设性** — 不仅仅是点评好坏，给出方向性判断或建议
- **客观性逻辑分析** — 摆数据、讲逻辑，避免主观情绪化表达
- **有深度** — 挖掘新闻背后的原因、影响、趋势，不浮于表面
- **每条约 60-120 字** — 足够展开分析，不过于冗长

**好的评论：**
> 央行选择此时降息，除了对冲外需走弱，更重要的是降低企业实际融资成本。但从历史经验看，降息传导至实体投资需要 3-6 个月，不能预期立竿见影。下半年的关键还是财政政策如何配合。

**不好的评论：**
> 降息是好事，能刺激经济。

### 评论分配建议

选评论的 8 条优先挑以下类型：
1. 政策变化类（经济/民生）— 分析影响面
2. 行业趋势类（科技/经济）— 判断方向
3. 国际关系类（时事）— 解读背后逻辑
4. 社会话题类（民生/娱乐）— 提供多角度思考

纯事实通报类（灾害通报、数据公布）可不加评论。

## 用法

```
/signal-pop-news-select path=filtered_news_20260717.json
```

skill 会自动识别日期，判断稿型，按规则输出。

## 可选参数

| 参数 | 说明 |
|------|------|
| `type=weekday` | 强制指定平日稿 |
| `type=weekend` | 强制指定周末稿 |
| `date=20260717` | 指定制作日期（覆盖自动识别） |
| `focus=科技` | 侧重某类别 |
| `exclude=某某公司` | 额外排除某关键词 |

## 注意事项

- 周末稿确保 15 条内容与本周平日稿不重复
- 类别分布按当天/当周热点灵活调整，不机械平均
- 如池中符合条件条目不足，按实际数量输出并说明
- 允许用户手动增删调整后再输出
