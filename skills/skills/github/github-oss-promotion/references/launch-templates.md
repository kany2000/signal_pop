## In-App Review Prompt (JavaScript)

```javascript
// Trigger after user successfully uses the product 5 times
let count = parseInt(localStorage.getItem('qt_usage_count') || '0', 10);
count++;
localStorage.setItem('qt_usage_count', count.toString());

if (count === 5 && !localStorage.getItem('qt_review_asked')) {
  showReviewPrompt({
    message: 'Enjoying QuickTranslate? Leave a review!',
    positiveBtn: '⭐ Rate on Web Store',
    dismissBtn: 'Not now'
  });
  localStorage.setItem('qt_review_asked', 'true');
}
// "Not now" should permanently dismiss the prompt
```

⚠️ **Critical**: YouTube descriptions CANNOT contain `<` or `>` characters — they're treated as HTML tags and silently stripped or cause errors. Use "under" or "less than" instead (e.g. `<500ms` → "under 500ms"). Timestamps only work for videos 25s+. For shorter videos, omit timestamps or use scene labels instead.
