# Meme Maker

## When to use  
User asks to make/create/generate a meme, reaction image, or funny caption image.

## How to use

### Method 1: ASCII Art (zero dep)
Print a framed ASCII meme with top/bottom text.

### Method 2: PIL Image  
Use Python Pillow to overlay text on an image.

### Method 3: imgflip API  
Call imgflip.com API to generate popular meme templates.
